Device & Network Restrictions
⛔
ห้ามใช้อุปกรณ์ IT ส่วนตัวเชื่อมต่อเข้ากับ ระบบโดเมนของบริษัท โดยเด็ดขาด
  • PC และ Server ที่เชื่อมต่อกับ TCPR Network จะต้องไม่เชื่อมต่อกับ Network อื่นนอกเหนือจากที่ได้รับอนุมัติจากแผนก IT
  • การใช้งาน Remote Access (Telework) ต้องผ่าน VPN Gateway เท่านั้น และต้องให้ผู้ดูแลระบบเป็นผู้ติดตั้ง VPN เท่านั้น
  • อุปกรณ์มือถือส่วนตัวหรือของบริษัทเชื่อมต่อระบบภายในได้เฉพาะ อีเมลเท่านั้น
  • ห้ามใช้อุปกรณ์สารสนเทศของบริษัทเชื่อมต่อกับ Public WiFi โดยเด็ดขาด
  • อุปกรณ์และทรัพย์สิน IT ของบริษัท ใช้ได้เพื่อ วัตถุประสงค์ทางธุรกิจเท่านั้น
💾
การจัดการข้อมูลและการสำรองข้อมูล
Data Management & Backup
✅
ระบบสำรองข้อมูล อัตโนมัติทุกวัน เก็บไว้ 31 วัน และ Backup ลงเทปทุกสิ้นเดือน
  • จัดเก็บข้อมูลสำคัญไว้ใน File Server ที่มีการสำรองข้อมูลเท่านั้น
  • หากต้องการกู้คืนข้อมูล ให้ยื่น IT Request ที่ http://tcpr-intranet พร้อมระบุชื่อไฟล์ โฟลเดอร์ และช่วงเวลาที่ใช้งานได้ล่าสุด
  • การส่งไฟล์สำคัญออกภายนอกผ่าน Email หรือ Cloud ต้องปฏิบัติตามขั้นตอน Encryption (17-C-008)
  • การนำข้อมูลสำคัญออกนอกสำนักงานต้องได้รับอนุญาตจาก ผู้ดูแลข้อมูลก่อน
  • เมื่อไม่ต้องการสิทธิ์การเข้าถึงแล้ว ต้องแจ้งผู้ดูแลระบบและ คืนสิทธิ์ทันที
💻
กฎการใช้คอมพิวเตอร์ส่วนบุคคล

a
ล็อกหน้าจอ ทุกครั้งที่ลุกออกจากที่นั่ง และปิดเครื่องเมื่อเลิกงาน
b

c

d
อย่าทิ้ง PC ไว้โดยไม่มีคนดูแลเมื่อ นำออกนอกสำนักงาน
e
การย้ายหรือทิ้ง PC ต้องแจ้ง IT Asset Manager ก่อนดำเนินการ
f
สื่อบันทึกข้อมูลควบคุมผ่านโปรแกรม IP Guard ที่ติดตั้งทุกเครื่อง

Email & Internet Usage Rules
🎣
หากได้รับ อีเมลน่าสงสัย ห้ามเปิด Attachment หรือคลิก URL และแจ้ง Admin ทันที
  • ใช้อีเมลที่กำหนดโดยแผนก IT เท่านั้น ห้ามใช้อีเมลส่วนตัวสำหรับงาน
  • ตรวจสอบ Email Address ของผู้รับก่อนส่งทุกครั้ง โดยเฉพาะข้อมูลสำคัญ
  • ห้าม Forward อีเมลอัตโนมัติไปยังภายนอก @tcp.co.th หากจำเป็นต้องปรึกษา IT
  • ใช้ Cloud สาธารณะที่ได้รับอนุมัติจาก IT เท่านั้น และปรึกษา Admin ก่อนใช้บริการ Cloud ใหม่
  • ห้ามโพสต์ข้อมูลธุรกิจลงบน Cloud สาธารณะที่ ไม่ระบุตัวตนผู้เข้าถึง
🚨
การรายงานเหตุการณ์ด้านความปลอดภัย
Security Incident Reporting
🆘
หาก PC หรือสื่อจัดเก็บข้อมูล สูญหาย/ถูกขโมย หรือพบไวรัส ให้แจ้ง Admin ทันที
  • ติดตั้ง Antivirus Software ที่กำหนดโดย IT และ อัปเดต Patch สม่ำเสมอ
  • ซอฟต์แวร์ทุกตัวต้องได้รับลิขสิทธิ์ถูกต้อง การติดตั้ง/ถอดถอนต้องยื่น IT Request
  • กิจกรรมของ PC ทั้งหมดถูกบันทึกโดยอัตโนมัติและจะถูกตรวจสอบตามความจำเป็น
  • เมื่อมีคำสั่งมาตรการฉุกเฉินจาก IT ให้ ร่วมมือและดำเนินการทันที
